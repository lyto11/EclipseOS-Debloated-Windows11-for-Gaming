## What's Removed / Disabled

- **Telemetry & tracking** — DiagTrack and related services fully disabled
- **Xbox services** — XblAuthManager, XblGameSave, XboxGipSvc, XboxNetApiSvc
- **OneDrive** — All OneDrive services and scheduled tasks removed
- **Windows Search** — WSearch disabled
- **Hyper-V & WSL** — All virtualization components disabled
- **Bluetooth, Fax, Print Spooler** — Disabled by default
- **SMB1 Protocol** — Disabled (security best practice)
- **Windows Update (wuauserv)** — Disabled
- **BITS** — Background Intelligent Transfer Service disabled
- **Connected Devices Platform** — CDPSvc and CDPUserSvc disabled

---

## What's Kept

- Core Windows services (networking, audio, storage, security)
- **DirectPlay** — enabled for gaming compatibility
- **LegacyComponents** — enabled for broader compatibility
- Hardware drivers (AMD, Intel, Realtek, etc.)
- Media Player & Notepad

---

## Post-Install Scripts

EclipseOS includes an automatic post-install system that runs on first boot:

- **Services** — Applies minimal or default service configurations via registry files
- **GPU** — GPU-specific optimization scripts
- **AntiCheat** — Anti-cheat configurations
- **Mitigations** — Security mitigation settings
- **Tweaks** — Additional system performance tweaks
